# `@justeattakeaway/pie-webc`

`@justeattakeaway/pie-webc` is variation of the [Just Eat Takeaway.com design system](https://pie.design/), PIE, using [Web Components](https://developer.mozilla.org/en-US/docs/Web/API/Web_components). Web Components allow us to build our components once and use them across
all of our web UIs regardless of the framework they have been build with. Our source code is open source, and can be found on [Github](https://github.com/justeattakeaway/pie).

## Getting started

How you set up our web components varies slightly depending on what framework your application is built with, but regardless of what tech stack you use, the first step will be installing our core packages into your application.
These packages are `@justeattakeaway/pie-webc` and `@justeattakeaway/pie-css`:

Using Yarn:

```bash
yarn add @justeattakeaway/pie-webc @justeattakeaway/pie-css
```
Using NPM:

```bash
npm install @justeattakeaway/pie-webc @justeattakeaway/pie-css
```

From here, you will need to set up Typography and base styles. Please read the [Typography and Styles](/docs/introduction-typography-and-styles--docs) page for more information.
For framework specific setup, please read our Integration guides in the sidebar.
